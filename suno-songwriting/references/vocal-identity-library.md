# Vocal Identity Library Index

Use this index before loading named fictional vocal identity detail. For ordinary singer direction, start with `vocal-routing-index.md` or `vocal-style-knowledge.md` instead. All profiles are voices intended for Suno prompt design and comparison testing.

## Files

- `vocal-feature-space.md`: coordinate dimensions, design assumptions, promptable vocal traits, and failure-mode categories.
- `classical-vocal-classification-source.md`: raw source table for choral categories, expanded classical voice types, Fach/subtypes, and technical terminology.
- `vocal-profile-compact-table.md`: fastest lookup for VP001-VP060 and compact prompt phrases.
- `vocal-profile-cards.md`: index for detailed profile-card cluster files with strengths, weaknesses, nearest neighbors, and listening notes.
- `vocal-profile-graph.md`: graph edges connecting similar-but-contrasted profiles.
- `vocal-comparison-song.md`: common test song and movement requirements for A/B voice comparison.
- `vocal-style-prompts-by-profile.md`: index for cluster files containing full Suno prompts by profile.
- `vocal-evaluation-rubric.md`: 1-5 scoring rubric.
- `vocal-identity-generation-instructions.md`: source instructions for creating more profile libraries.

## Routing

- Need a distinctive voice phrase from the profile set: load `vocal-profile-compact-table.md`.
- Need coordinate dimensions or safe non-imitative trait design: load `vocal-feature-space.md` or `vocal-style-knowledge.md`.
- Need classical, choral, SATB, or Fach terminology: load `classical-vocal-classification-source.md`.
- Need one named profile in depth: load `vocal-profile-cards.md`, identify the VP cluster, then load only that `vocal-profiles-*.md` cluster file and search for the VP id or profile name.
- Need comparison testing: load `vocal-comparison-song.md`, `vocal-profile-compact-table.md`, and optionally `vocal-evaluation-rubric.md`.
- Need nearest-neighbor contrast: load `vocal-profile-graph.md`.
- Need profile-specific Suno prompts: load `vocal-style-prompts-by-profile.md`, identify the VP cluster, then load only that `vocal-prompts-*.md` cluster file.

## Profile Clusters

- VP001-VP006: studio pop/R&B colorists.
- VP007-VP012: rock, theater, and metal force voices.
- VP013-VP018: jazz, soul, and gospel interpreters.
- VP019-VP024: folk, country, and roots narrators.
- VP025-VP030: electronic and experimental processors.
- VP031-VP036: groove and dance-call voices.
- VP037-VP042: devotional, classical, and chamber hybrids.
- VP043-VP048: diction-forward regional color profiles.
- VP049-VP054: edge and extreme hybrid voices.
- VP055-VP060: cross-cultural process voices.
