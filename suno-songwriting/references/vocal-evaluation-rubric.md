# Vocal Evaluation Rubric

Scoring rubric for evaluating generated vocal-profile outputs.

## IX. Evaluation rubric

Use a 1–5 scale for each category. Score against the intended profile, not against generic vocal quality.

| Category | 1 | 2 | 3 | 4 | 5 |
|---|---|---|---|---|---|
| Perceived vocal identity distinctness | Sounds generic or same as other outputs | Mild differences only | Some clear identity markers | Strongly identifiable profile | Immediately distinct and memorable |
| Timbral distinctness | Timbre not aligned with prompt | Weak or inconsistent color | Acceptable color in key sections | Strong color most of the song | Timbral identity is unmistakable throughout |
| Range/register behavior | Ignores range/tessitura instructions | Occasional register hints | Register behavior partly audible | Register choices mostly correct | Range and register define the performance |
| Diction/accent audibility | Diction unclear or caricatured | Diction cue barely present | Diction present but uneven | Natural and audible | Respectful, clear, musically integrated |
| Genre adherence | Arrangement/vocal style mismatched | Some genre elements but unstable | Genre generally recognizable | Genre supports vocal profile well | Genre and vocal identity are mutually reinforcing |
| Emotional delivery | Emotion absent or wrong | Weak emotional stance | Adequate emotional reading | Convincing stance | Emotion feels specific, embodied, and profile-native |
| Section-by-section movement compliance | Ignores song movements | Only one or two movements audible | Half the sections comply | Most sections clearly shaped | Each section reveals intended behavior |
| Lyric intelligibility | Lyrics largely unintelligible | Many words lost | Main hook clear, verses uneven | Mostly clear | Clear throughout, even in ornament/processing |
| Stability/consistency across song | Voice changes identity randomly | Frequent unstable artifacts | Some consistency issues | Mostly stable | Stable fictional identity from intro to outro |
| Uniqueness vs nearest neighbors | Indistinguishable from neighbors | Minor difference only | Some contrast audible | Clearly different from neighbors | Difference is obvious on multiple axes |
| Prompt usefulness/reproducibility | Prompt produces chaotic results | Results inconsistent | Usable with retries | Reliable profile behavior | Repeatably produces intended vocal identity |

Suggested scoring workflow:
1. Generate the same lyrics with 3–5 candidate profiles.
2. Blind-label outputs if possible.
3. Score each category 1–5.
4. Specifically compare each output against its two nearest neighbors.
5. Revise prompts that score below 3 in distinctness, intelligibility, or stability.

## X. Comparison Notes

- These profiles are fictional design coordinates for comparing vocal, style, arrangement, and movement behavior.
- Keep references at the level of musical traits: timbre color, range, vibrato type, register behavior, genre gravity, production texture, rhythmic feel, and emotional stance.
- Treat accents and dialect cues as musical articulation variables: vowel openness, consonant crispness, rhythmic speech influence, diction foregrounding, and broad regional color.
- All profiles are adult performers.
- Preserve identical lyrics across tests so that perceived differences can be attributed to voice, style, arrangement, and movement behavior rather than lyric changes.
