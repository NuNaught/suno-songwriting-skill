# Vocal Routing Index

Use this first for vocal direction. Most Suno prompts only need a compact vocal phrase; avoid loading the full vocal identity library unless the user asks for named fictional profiles, comparison testing, or detailed voice design.

## Fast Routes

- Need an ordinary style prompt voice phrase: use `vocal-style-knowledge.md`, especially the coordinate workflow and prompt assembly sections.
- Need a compact singer descriptor from a profile list: use `vocal-profile-compact-table.md`.
- Need detailed coordinate vocabulary or safe conversion from artist references: use `vocal-feature-space.md`.
- Need classical, choral, SATB, opera, Fach, or technical range terminology: use `classical-vocal-classification-source.md`.
- Need bracketed vocal delivery tags for the lyric field: use `lyric-tag-knowledge.md`; for larger tag catalogs, route through `structure-movement-tags.md`.
- Need a named vocal profile, profile comparison, profile-specific prompt, or evaluation rubric: use `vocal-identity-library.md`.

## Common Lightweight Phrases

Use these as building blocks when the user did not request deep vocal design:

- `smoky alto, close-mic delivery, soft consonants, delayed vibrato`
- `bright tenor, clean pop diction, smooth mix voice, polished harmonies`
- `warm baritone storyteller, relaxed phrasing, light rasp, intimate room`
- `gospel-soul lead, open-throat chorus lift, melismatic touches, choir responses`
- `indie-folk duet, conversational delivery, natural harmonies, minimal processing`
- `rap lead, crisp consonants, confident pocket, sparse ad-libs`
- `chamber soprano, clear head voice, restrained vibrato, reverent tone`
- `rock mezzo, controlled grit, chesty chorus lift, live-band presence`

## Escalation Rules

- Stay lightweight for prompts like "female indie vocal," "male R&B singer," "choir," "spoken verse," or "rap feature."
- Use `vocal-style-knowledge.md` when the request needs 5-8 coordinated traits or section-by-section vocal movement.
- Use `vocal-feature-space.md` when translating a reference into concrete vocal traits.
- Use `vocal-identity-library.md` only when the user asks for profiles, VP ids, profile cards, A/B comparison, nearest neighbors, full profile prompts, or scoring.
- Do not combine multiple vocal cluster files unless the user explicitly compares profiles across clusters.
