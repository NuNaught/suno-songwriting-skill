# Extended Genre Diversity Matrix Index

Use this index before loading genre-family files. For common genres, prefer `genre-knowledge.md`; load these files when the user needs uncommon, historical, regional, fusion, or instrument-specific genre vocabulary.

## Files

- `genres-pop-rock-mainstream.md`: modern pop, rock, R&B, idol-pop, country-pop, and mainstream radio lanes.
- `genres-electronic-club.md`: house, techno, trance, bass music, hyperpop, future bass, EDM festival pop, and beat-based electronic styles.
- `genres-hiphop-urban.md`: trap, drill, lo-fi hip-hop, baile funk, Jersey club, dembow, reggaeton, and MC-driven beat forms.
- `genres-latin-caribbean.md`: reggaeton, dembow, cumbia, son, samba, frevo, tango, fado, flamenco, Latin pop, bachata, and Andean forms.
- `genres-african-diasporic.md`: gnawa, rai, mbaqanga, highlife, kwaito, Afrobeat, Afrobeats, amapiano, taarab, benga, and diasporic carnival/favela grooves.
- `genres-classical-liturgical.md`: plainchant, Ars subtilior, madrigal, fugue, waltz, minimalism, and third stream.
- `genres-jazz-blues-roots.md`: Delta blues, bebop, bluegrass, zydeco, fiddle traditions, klezmer, country-pop, and singer-songwriter/adult contemporary lanes.
- `genres-global-folk-devotional.md`: qawwali, dhrupad, gagaku, Noh, gamelan, throat singing, Bulgarian choir, Iberian/Andean/regional folk, and devotional traditions.

## Routing

- Need mainstream prompt vocabulary: load `genres-pop-rock-mainstream.md`.
- Need club or electronic production: load `genres-electronic-club.md`.
- Need rap or beat architecture: load `genres-hiphop-urban.md`.
- Need Spanish, Portuguese, Caribbean, or Latin American references: load `genres-latin-caribbean.md`.
- Need African or diasporic groove systems: load `genres-african-diasporic.md`.
- Need historical art music or liturgical language: load `genres-classical-liturgical.md`.
- Need acoustic roots, jazz, blues, or country storytelling: load `genres-jazz-blues-roots.md`.
- Need devotional, ritual, or global folk/classical traditions: load `genres-global-folk-devotional.md`.
