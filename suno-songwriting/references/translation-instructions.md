# Translation Instructions

For song translation and multilingual lyrics, preserve singability, emotional intent, and section function.

## Translation Priorities

1. Meaning and emotional arc
2. Natural phrasing in the target language
3. Singable rhythm and stress
4. Rhyme or sonic cohesion
5. Literal wording

## Workflow

- Identify source language, target language, genre, and whether the translation must match melody.
- If matching an existing melody, preserve syllable counts approximately and keep stressed words in strong positions.
- If writing fresh lyrics in another language, localize idioms rather than translating them literally.
- For bilingual songs, decide which language owns the hook and which language carries narrative details.

## Bilingual Patterns

- English verses with Spanish chorus for hook emphasis.
- Native-language verses with English post-chorus chant.
- Alternating duet perspectives by language.
- Code-switching inside lines for conversational realism.

## Output Notes

- If useful, include a brief note explaining non-literal choices.
- Do not over-explain when the user only needs paste-ready lyrics.
- Preserve bracketed section tags in English unless the user requests localized tags.

## Guardrails

- Avoid machine-translation stiffness.
- Avoid false rhymes created only by awkward syntax.
- Use culturally natural imagery for the target language and audience.
